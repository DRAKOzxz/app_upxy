## 🔥 ¿Qué es app_upxy?

**app_upxy** es una aplicación web que permite:

📁 Compartir archivos  
💬 Chatear en tiempo real  
🔐 Gestionar usuarios  

Todo dentro de tu red local, sin depender de servicios externos.

---

## 🚀 Nuevas mejoras

✨ Sistema completo de autenticación  
💬 Chat global persistente  
🔐 Hash de contraseñas  
🗄️ Base de datos SQLite  

---

## 📦 Requisitos

Antes de instalar, asegúrate de tener:

- 🐍 Python 3.10+
- 🔧 Git instalado
- 🖥️ Terminal (recomendado)

### 🐉 Opcional (entorno alternativo)

Si tienes problemas en Windows:

- Usa consola de **Kali Linux (WSL)**
- O instala **Win-KeX** para entorno gráfico

---

## ⚡ Instalación

```bash
# Clonar repositorio
git clone https://github.com/DRAKOzxz/app_upxy.git

cd app_upxy

# Crear entorno virtual
python -m venv .venv

# Activar entorno
source .venv/bin/activate   # Linux / Mac / Kali
.venv\Scripts\activate      # Windows

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar app
python app.py
