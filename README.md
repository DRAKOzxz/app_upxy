## ✨ Descripción

**FileShare App** es una aplicación diseñada para permitir a los usuarios:

🔹 Subir archivos  
🔹 Ver contenido compartido  
🔹 Descargar archivos  
🔹 Compartir con otros usuarios  

Todo en una plataforma simple, rápida y accesible.
